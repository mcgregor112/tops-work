q - Sorting and reversing a list using sort(), sorted(), and reverse().

Answer :

Certainly, here's a theoretical explanation of sorting and reversing lists in Python:

1. Sort() :

Sorting involves arranging the elements of a list in a specific order, typically ascending or descending.
sort(): This method modifies the original list itself, rearranging its elements in the specified order (ascending by default).

2. sorted(): This method creates a new, sorted list while leaving the original list unchanged. This is often preferred when you need to preserve the original list.

3. reverse() : This method modifies the original list by reversing the order of its elements.
