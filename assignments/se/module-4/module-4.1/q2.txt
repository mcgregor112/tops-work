q. What is oop? List OOP concept.

Answer -

- OOP stands for " Object oriented programming " , it is  a programming paradigm that organizes programs around objects instead of functions and logic. In C++, OOP is implemented using a variety of concepts.

- lists of OOP :

1. Class
2. Object
3. Encapsulation
4. Abstraction
5. Polymorphism
6. Inheritance
7. Dynamic Binding
8. Message Passing
